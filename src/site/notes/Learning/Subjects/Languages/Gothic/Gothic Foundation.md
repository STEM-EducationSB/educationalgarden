## 4. Nouns & Cases (Minimal Survival Set)

Gothic uses **4 cases**:

|Case|Meaning|
|---|---|
|Nominative|subject|
|Accusative|object|
|Genitive|of|
|Dative|to / for|

Example noun: **guma** (man)

|Case|Singular|
|---|---|
|Nom|guma|
|Acc|guman|
|Gen|gumins|
|Dat|gumin|

You do **not** guess endings. You memorize patterns.
