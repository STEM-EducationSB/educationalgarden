## 1. Personal Pronouns (Subject)

|English|Dutch|Notes|
|---|---|---|
|I|**ik**|Always lowercase|
|you (sing., informal)|**jij / je**|_je_ is unstressed|
|you (sing., formal)|**u**|Used with strangers, elders|
|he|**hij**||
|she|**zij / ze**|_ze_ is unstressed|
|it|**het**|Refers to _het-words_|
|it|**hij / zij**|For _de-words_|
|we|**wij / we**|_we_ is unstressed|
|you (plural)|**jullie**|Informal plural|
|they|**zij / ze**|Context decides meaning|

---

## 2. Object Pronouns

|English|Dutch|
|---|---|
|me|**mij / me**|
|you (inf.)|**jou / je**|
|you (formal)|**u**|
|him|**hem**|
|her|**haar**|
|it|**het**|
|us|**ons**|
|you (pl.)|**jullie**|
|them|**hen / ze**|

**Rule:**

- **mij, jou, hem, haar** → stressed
    
- **me, je, ’m, d’r, ze** → unstressed
    

---

## 3. Possessive Pronouns (Before Noun)

|English|Dutch|
|---|---|
|my|**mijn**|
|your (inf.)|**jouw / je**|
|your (formal)|**uw**|
|his|**zijn**|
|her|**haar**|
|its|**zijn / haar**|
|our|**ons / onze**|
|your (pl.)|**jullie**|
|their|**hun**|

**ons vs onze**

- **ons** → het-word
    
- **onze** → de-word

## Blunt Reality Check

Dutch pronouns are **easy to memorize** but **hard to master** because:

- unstressed forms dominate real speech
    
- gender depends on **de/het**, not meaning
    
- word order punishes sloppy usage
