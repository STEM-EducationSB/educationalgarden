# **1. Subject Pronouns (Used to say _who_ is doing the action)**

These are the “I / you / he / she / we / they” forms.

|Spanish|Meaning|Notes|
|---|---|---|
|**yo**|I|Easy.|
|**tú**|you (informal, singular)|Use with friends, people your age.|
|**usted (Ud.)**|you (formal, singular)|Respectful.|
|**él**|he|Accent matters.|
|**ella**|she|—|
|**nosotros**|we (all male or mixed)|—|
|**nosotras**|we (all female)|—|
|**vosotros**|you all (informal, plural)|Used mainly in Spain.|
|**vosotras**|you all (informal, plural, female)|Spain only.|
|**ustedes (Uds.)**|you all (plural)|Used everywhere; replaces “vosotros” in Latin America.|
|**ellos**|they (male/mixed)|—|
|**ellas**|they (female)|—|

If you’re learning fast for communication, focus on:  
**yo, tú, él, ella, nosotros, ustedes, ellos**

---

# **2. Direct Object Pronouns (Who/what receives the action)**

Used with verbs like _see, know, love, want._

|Spanish|Meaning|
|---|---|
|**me**|me|
|**te**|you|
|**lo / la**|him, it (masc) / her, it (fem)|
|**nos**|us|
|**os**|you all (Spain)|
|**los / las**|them|

Example:  
**Yo te veo.**  
I see you.

**Ella lo tiene.**  
She has it.

---

# **3. Indirect Object Pronouns (To whom / for whom)**

Used with verbs like _give, tell, say, show._

|Spanish|Meaning|
|---|---|
|**me**|to/for me|
|**te**|to/for you|
|**le**|to/for him/her/you|
|**nos**|to/for us|
|**os**|to/for you all|
|**les**|to/for them|

Example:  
**Yo le doy el libro.**  
I give the book _to him/her._

---

# **4. Reflexive Pronouns (When the action comes back to the subject)**

Used with verbs like _wash, get up, call oneself._

|Spanish|Meaning|
|---|---|
|**me**|myself|
|**te**|yourself|
|**se**|himself/herself/yourself|
|**nos**|ourselves|
|**os**|yourselves|
|**se**|themselves|

Example:  
**Me levanto.**  
I get up.

**Se llama Carlos.**  
He calls himself Carlos.

---

# **5. Possessive Adjectives (My / your / his / our / their)**

These come _before_ nouns.

|Spanish|Meaning|
|---|---|
|**mi / mis**|my|
|**tu / tus**|your (informal)|
|**su / sus**|his/her/their/your (formal)|
|**nuestro / nuestra**|our|
|**nuestros / nuestras**|our (plural)|

Example:  
**Mi casa.** (my house)  
**Nuestros amigos.** (our friends)

---

# **6. Possessive Pronouns (mine / yours / his...)**

These replace nouns.

|Spanish|Meaning|
|---|---|
|**mío / mía / míos / mías**|mine|
|**tuyo / tuya / tuyos / tuyas**|yours|
|**suyo / suya / suyos / suyas**|his/hers/yours|
|**nuestro / nuestra**|ours|
|**vuestro / vuestra**|yours (Spain)|

Example:  
**La casa es mía.**  
The house is mine.

---

# **7. Demonstrative Pronouns (this / that / these / those)**

|Spanish|Meaning|
|---|---|
|**este / esta**|this|
|**estos / estas**|these|
|**ese / esa**|that|
|**esos / esas**|those|
|**aquel / aquella**|that (far away)|
|**aquellos / aquellas**|those (far away)|

Example:  
**Este es mío.**  
This one is mine.

---

# **8. Relative Pronouns (that / who / which)**

|Spanish|Meaning|
|---|---|
|**que**|that, who, which|
|**quien / quienes**|who (sing/pl)|
|**cual / cuales**|which|
|**donde**|where|

Example:  
**El hombre que vive aquí.**  
The man _who_ lives here.
