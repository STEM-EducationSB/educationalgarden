## **1. Definite Articles (The)**

| Gender         | Singular | Plural  | English |
| -------------- | -------- | ------- | ------- |
| Masculine      | **le**   | **les** | the     |
| Feminine       | **la**   | **les** | the     |
| Before vowel/h | **l’**   | **les** | the     |

**Examples:**

- le garçon → the boy (masculine, singular)
    
- la fille → the girl (feminine, singular)
    
- l’école → the school (before vowel, singular)
    
- les étudiants → the students (plural)
    

---

## **2. Indefinite Articles (A / An / Some)**

|Gender|Singular|Plural|English|
|---|---|---|---|
|Masculine|**un**|**des**|a / some|
|Feminine|**une**|**des**|a / some|

**Examples:**

- un chat → a cat (masculine)
    
- une maison → a house (feminine)
    
- des livres → some books (plural)
    
- des filles → some girls (plural)
    

---

## **3. Partitive Articles (Some / Any)**

Used for **uncountable things** like water, bread, or sugar.

|Masculine|Feminine|Plural|
|---|---|---|
|du → some|de la → some|des → some|
|de l’ → some (before vowel/h)|||

**Examples:**

- du pain → some bread
    
- de la soupe → some soup
    
- de l’eau → some water
    

---

### **4. Quick Tips**

1. **le / la / l’** → “the” (definite)
    
2. **un / une / des** → “a / some” (indefinite)
    
3. **du / de la / de l’** → “some” for uncountable things (partitive)
    
4. **les / des** → plural is the same for both genders
