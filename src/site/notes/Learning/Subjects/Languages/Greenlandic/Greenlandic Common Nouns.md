# Greenlandic Common Nouns

**Greenlandic word – “approx. pronunciation” – /IPA/ – English meaning**

Notes you should know upfront (telling it like it is):

- Greenlandic pronunciation is **highly phonetic**, but **long words + geminate consonants** make perfect IPA difficult even for linguists.
    
- The IPA given here is **standard academic approximation**, not casual guessing.
    
- Greenlandic does **not** use articles like _the / a_; English gloss uses “the” for clarity only.
    

This list is optimized for **learning + memorization**, not poetic overreach.
