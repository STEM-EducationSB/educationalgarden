# CORE IDEA (Very Important)

Every Italian verb works like this:

> **INFINITIVE → CUT → ADD ENDING (based on tense + pronoun)**

So we always need **3 things**:

1. Verb type (-ARE / -ERE / -IRE / -ISC)
    
2. Tense
    
3. Pronoun
    

If you know the **ending table**, you can conjugate **any verb instantly**.

---

# STEP 1 — SUBJECT PRONOUNS (Fixed Forever)

Memorize this ONCE. Never changes.

|Person|Pronoun|
|---|---|
|1st sg|io|
|2nd sg|tu|
|3rd sg|lui / lei|
|1st pl|noi|
|2nd pl|voi|
|3rd pl|loro|

Everything you do later just **attaches endings** to these rows.

---

# STEP 2 — VERB TYPES (Infinitive Recognition)

Look only at the **last 3 letters**:

|Infinitive|Type|Example|
|---|---|---|
|-are|ARE|parlare|
|-ere|ERE|credere|
|-ire|IRE|dormire|
|-ire + isc|ISC-IRE|finire|

Rule:  
• **Cut the final -re**  
• What remains is your **base**

Examples:

- parlare → parla-
    
- credere → crede-
    
- dormire → dormi-
    
- finire → fini-
    

---

# STEP 3 — PRESENT TENSE (Presente)

This is your **foundation tense**.

### How to apply

1. Take infinitive
    
2. Remove **-are / -ere / -ire**
    
3. Add ending from the table
    

---

## PRESENT TENSE ENDINGS TABLE

### **-ARE verbs**

| Pronoun | Ending |
| ------- | ------ |
| io      | -o     |
| tu      | -i     |
| lui/lei | -a     |
| noi     | -iamo  |
| voi     | -ate   |
| loro    | -ano   |

**Formula:**  
parl + o → io parlo

---

### **-ERE verbs**

| Pronoun | Ending |
| ------- | ------ |
| io      | -o     |
| tu      | -i     |
| lui/lei | -e     |
| noi     | -iamo  |
| voi     | -ete   |
| loro    | -ono   |

---

### **-IRE verbs (normal)**

|Pronoun|Ending|
|---|---|
|io|-o|
|tu|-i|
|lui/lei|-e|
|noi|-iamo|
|voi|-ite|
|loro|-ono|

---

### **-IRE (ISC verbs)**

**Rule:** add **-isc-** in 3 forms only  
(io, tu, lui/lei, loro)

|Pronoun|Ending|
|---|---|
|io|-isco|
|tu|-isci|
|lui/lei|-isce|
|noi|-iamo|
|voi|-ite|
|loro|-iscono|

---

### HOW TO MEMORIZE PRESENT

• **io = o**  
• **tu = i**  
• **noi = iamo (always)**  
• **voi = te/ta/ti**  
• **loro = no**

---

# STEP 4 — IMPERFECT TENSE (Imperfetto)

This is the **easiest tense in Italian**.

### Golden Rule

> **Same endings for ALL verbs**

### Formula

infinitive – **re** + ending

---

## IMPERFECT ENDINGS (ALL VERBS)

|Pronoun|Ending|
|---|---|
|io|-vo|
|tu|-vi|
|lui/lei|-va|
|noi|-vamo|
|voi|-vate|
|loro|-vano|

Example:

- parlare → parlavo
    
- credere → credevo
    
- dormire → dormivo
    
- finire → finivo
    

### Memory Trick

Every form has **V**  
If you see **V**, think **imperfect**

---

# STEP 5 — FUTURE SIMPLE (Futuro)

Very systematic.

### Formula

infinitive – **e** + future ending

---

## FUTURE ENDINGS (ALL VERBS)

|Pronoun|Ending|
|---|---|
|io|-ò|
|tu|-ai|
|lui/lei|-à|
|noi|-emo|
|voi|-ete|
|loro|-anno|

Example:

- parlare → parlerò
    
- credere → crederai
    
- dormire → dormiranno
    

### Memory Pattern

ò – ai – à – emo – ete – anno

---

# STEP 6 — CONDITIONAL PRESENT

Built from **future stem**.

### Formula

future stem + conditional endings

---

## CONDITIONAL ENDINGS (ALL VERBS)

|Pronoun|Ending|
|---|---|
|io|-ei|
|tu|-esti|
|lui/lei|-ebbe|
|noi|-emmo|
|voi|-este|
|loro|-ebbero|

Example:

- parlare → parlerei
    
- credere → crederei
    

---

# STEP 7 — PAST PARTICIPLE (Key for all compound tenses)

Just memorize **3 endings**:

|Verb type|Ending|
|---|---|
|-are|-ato|
|-ere|-uto|
|-ire|-ito|

---

# STEP 8 — PASSATO PROSSIMO (Compound Past)

### Formula

**pronoun + avere/essere (present) + participle**

You do **NOT conjugate the main verb** here.

Example structure:

- io **ho** parlato
    
- noi **abbiamo** dormito
    
- lei **è** andata
    

---

# STEP 9 — CONGIUNTIVO (Pattern-Based, Not Hard)

## Congiuntivo Presente — ENDINGS ONLY

|Verb type|Singular|Plural|
|---|---|---|
|-ARE|-i|-iamo / -iate / -ino|
|-ERE|-a|-iamo / -iate / -ano|
|-IRE|-a|-iamo / -iate / -ano|

### Rule to memorize

ARE → **I**  
ERE/IRE → **A**

---

# HOW TO MEMORIZE EVERYTHING (Critical Advice)

### 1. Learn **endings**, not verbs

Italian has **few endings**, many verbs.

### 2. Memorize by **columns**, not rows

Example:

- io: o / vo / ò / ei
    
- tu: i / vi / ai / esti
    

### 3. Practice with ONE verb per type

parlare – credere – dormire – finire

### 4. Never memorize full tables

Memorize **patterns**, apply live.
