# VERB BASE (INFINTIVE EQUIVALENT)

**كتب – katab – to write**  
(Past, 3rd person masculine singular)

---

# TENSE 1 — PAST (Perfect)

## Formation Rule

**PAST STEM (katab-) + SUFFIX**

|Person (EN)|Person (AR)|Form|Transliteration|What Ending To Add With STEM|
|---|---|---|---|---|
|I|أنا (ana)|كتبت|katabt|ـت (-t)|
|You (m)|إنتَ (enta)|كتبت|katabt|ـت (-t)|
|You (f)|إنتِ (enti)|كتبتي|katabti|ـتي (-ti)|
|He|هو (huwwé)|كتب|katab|∅ (no ending)|
|She|هي (hiyye)|كتبت|katabet|ـت (-et)|
|We|نحنا (naḥna)|كتبنا|katabna|ـنا (-na)|
|You (pl)|إنتو (ento)|كتبتوا|katabtu|ـتو (-tu)|
|They|هنن (hennen)|كتبوا|katabu|ـوا (-u)|

## Example

1. أنا كتبت الرسالة  
    ana katabt er-risāle  
    I wrote the letter
    
2. هو كتب الاسم  
    huwwé katab el-esem  
    He wrote the name
    
3. هي كتبت الجواب  
    hiyye katabet el-jawāb  
    She wrote the answer
    
4. نحنا كتبنا التقرير  
    naḥna katabna et-taqrīr  
    We wrote the report
    
5. هنن كتبوا كل شي  
    hennen katabu kill شي  
    They wrote everything

# TENSE 2 — PRESENT / IMPERFECT

## Formation Rule

**PREFIX + PRESENT STEM (ktob)**

|Person (EN)|Person (AR)|Form|Transliteration|What Ending To Add With STEM|
|---|---|---|---|---|
|I|أنا (ana)|بكتب|baktob|بـ (ba-)|
|You (m)|إنتَ (enta)|بتكتب|btaktob|بتـ (bta-)|
|You (f)|إنتِ (enti)|بتكتبي|btaktbi|بتـ + ي (bta- + i)|
|He|هو (huwwé)|بيكتب|byaktob|بيـ (bya-)|
|She|هي (hiyye)|بتكتب|btaktob|بتـ (bta-)|
|We|نحنا (naḥna)|منكتب|mnaktob|منـ (mna-)|
|You (pl)|إنتو (ento)|بتكتبوا|btaktbu|بتـ + وا (bta- + u)|
|They|هنن (hennen)|بيكتبوا|byaktbu|بيـ + وا (bya- + u)|

## Example

1. أنا بكتب كل يوم  
    ana baktob kil yōm  
    I write every day
    
2. هو بيكتب بسرعة  
    huwwé byaktob bisurʿa  
    He writes fast
    
3. هي بتكتب كتير  
    hiyye btaktob ktīr  
    She writes a lot
    
4. نحنا منكتب سوا  
    naḥna mnaktob sawa  
    We write together
    
5. هنن بيكتبوا أسماء  
    hennen byaktbu asmāʼ  
    They write names

# TENSE 4 — FUTURE

**رح / حَ + PRESENT FORM**

|Person (EN)|Person (AR)|Form|Transliteration|What Ending To Add With STEM|
|---|---|---|---|---|
|I|أنا (ana)|رح أكتب|raḥ aktob|رح + بـ|
|You (m)|إنتَ (enta)|رح تكتب|raḥ tكتب|رح + بتـ|
|You (f)|إنتِ (enti)|رح تكتبي|raḥ tكتbi|رح + بتـ + ي|
|He|هو (huwwé)|رح يكتب|raḥ yaktob|رح + بيـ|
|She|هي (hiyye)|رح تكتب|raḥ taktob|رح + بتـ|

|You (pl)|إنتو (ento)|رح تكتبوا|raḥ tكتبوا|رح + بتـ + وا|
|They|هنن (hennen)|رح يكتبوا|raḥ yaktbu|رح + بيـ + وا|

## Example

1. أنا رح أكتب بكرا  
    ana raḥ aktob bukra

2. هو رح يكتب التقرير  
    huwwé raḥ yaktob et-taqrīr  
    He will write the report
    
3. هي رح تكتب رسالة  
    hiyye raḥ taktob risāle  
    She will write a letter
    
4. نحنا رح نكتب كل شي  
    naḥna raḥ naktob kill شي  
    We will write everything
    
5. هنن رح يكتبوا الأسماء  
    hennen raḥ yaktbu el-asmāʼ  
    They will write the names
